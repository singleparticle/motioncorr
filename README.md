MOTIONCORR UTILITY
============

Summary
-------
This motioncorr excutable utility run on BCM cluster.

# Background
## motioncorr
Corrects whole frame image motion recorded with dose fractionated image stack. MotionCorr is a bioinformatic software 
that greatly enhances image quality and data acquisition efficiency - key bottlenecks in applying near atomic resolution 
electron cryomicroscopy (cryoEM) to a broad range of protein samples.

[Visit motioncorr Website](http://cryoem.ucsf.edu/software/driftcorr.html)
